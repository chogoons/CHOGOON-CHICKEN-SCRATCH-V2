# 조군 개발새발 V2(CHOGOON CHICKEN SCRATCH V2)

조군 개발새발 V2(CHOGOON CHICKEN SCRATCH V2)는 조군(CHOGOON)이 제작한 한글 폰트입니다. 이 폰트는 "둘기마요" 폰트를 기반으로 한글(11,172자), 영문(52자), 숫자(10자) 및 일부 특수 문자를 수정하여 제작되었습니다.

## 특징
- **스타일**: 캐주얼하고 독특한 손글씨 스타일
- **지원 문자**: 한글 11,172자, 영문 52자, 숫자 10자, 특수 문자
- **파일 형식**: TTF, OTF
- **라이선스**: SIL Open Font License, Version 1.1
